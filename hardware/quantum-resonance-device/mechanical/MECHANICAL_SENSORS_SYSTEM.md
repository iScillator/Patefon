# 🎛️ Mechanical Sensors System - Система Механических Датчиков

## Принципы механического детектирования

Полностью механическая система датчиков основана на **классических физических принципах**:

### 🌊 Методы механического детектирования:
1. **Маятниковые системы** — детектирование вибраций и наклонов
2. **Резонансные пластины** — улавливание звуковых колебаний
3. **Барометрические мехи** — реакция на изменения давления
4. **Механические интерферометры** — измерение микро-смещений
5. **Струнные тензометры** — регистрация деформаций

## 🎯 Детекторы движения людей

### Маятниковые сейсмографы (8 шт):

```
МАЯТНИКОВЫЙ СЕЙСМОГРАФ:

Принцип работы:
- Тяжелый маятник (5 кг) подвешен на тонкой проволоке
- При вибрации пола маятник остается неподвижным в пространстве
- Основание движется относительно маятника
- Движение передается на записывающий механизм

Технические характеристики:
├── Масса маятника: 5 кг (свинец в латунном корпусе)
├── Длина подвеса: 1000 мм (стальная проволока Ø0.3 мм)
├── Собственная частота: ~0.5 Гц
├── Чувствительность: 0.01 мм смещения
├── Диапазон измерений: ±10 мм
├── Демпфирование: масляный амортизатор
└── Запись: игла на закопченной ленте

Схема конструкции:
     ┌─ Точка подвеса (карданный шарнир)
     │
     │ ← Стальная проволока (1000 мм)
     │
     ● ← Маятник (латунь + свинец, 5 кг)
     │
     │ ← Записывающая игла
     │
┌────┴────┐ ← Барабан с закопченной лентой
│ ░░░░░░░ │   (вращение от часового механизма)
│ ░░░░░░░ │
└─────────┘
     ▲
   Основание (связано с полом)
```


```
МЕТАЛЛИЧЕСКИЕ РЕЗОНАНСНЫЕ ПЛАСТИНЫ:

Набор из 88 пластин (как клавиши рояля):
├── Материал: закаленная сталь
├── Диапазон частот: 27.5 Гц (A0) - 4186 Гц (C8)
├── Размеры: от 200×50×3 мм до 50×10×1 мм
├── Настройка: точная обработка на ±1 цент
├── Крепление: на резонансных блоках
└── Регистрация: оптическое считывание колебаний

Принцип детектирования:
1. Каждая пластина настроена на определенную частоту
2. При звучании этой частоты пластина входит в резонанс
3. Амплитуда колебаний регистрируется механически
4. Создается спектральная картина звука в помещении

Механизм регистрации колебаний:
     Резонансная пластина
            │
            ▼
    ┌─══════════════─┐ ← Колеблющаяся пластина
    │                │   (сталь, настроена на
    │                │    определенную частоту)
    └─┬────────────┬─┘
      │            │
      ● ← Точки опоры
      │
      ├─ Рычаг передачи (1:10)
      │
      ▼
  ┌───────┐ ← Увеличенное движение
  │ ИГЛА  │   записывается на барабан
  └───────┘
```

```

├── Анероидная капсула: латунь, диаметр 60 мм

├── Точность: ±0.1 гПа

- Дополнительная игла записывает изменения

Влияние на звук:
- Изменения атмосферного давления влияют на скорость звука
- Плотность воздуха меняет резонансные частоты
- Система автоматически корректирует настройки
```

## 🔗 Система передачи данных

### Механическая обработка сигналов:

```
ЦЕНТРАЛЬНЫЙ ВЫЧИСЛИТЕЛЬНЫЙ МЕХАНИЗМ:

Аналоговый компьютер на основе:
├── Дифференциальных механизмов (как в счетных машинах)
├── Планиметров для интегрирования сигналов
├── Гармонических анализаторов для спектрального анализа
├── Механических сумматоров и умножителей
└── Кулачковых механизмов для нелинейных функций

Обработка включает:
1. Суммирование сигналов с датчиков
2. Вычисление средних значений за период
3. Анализ трендов и изменений
4. Принятие решений по алгоритмам
5. Выдача управляющих воздействий

Схема передачи данных:
    Датчики → Рычаги → Валы → Шестерни → Кулачки → Исполнительные механизмы
                         ↓
                   Центральный
                   вычислитель
                   (механический)
```


## 🔧 Калибровка и настройка

### Процедура начальной настройки:

```bash
1. Установка и выравнивание всех датчиков:
   - Проверка горизонтали маятников (уровень)
   - Настройка резонансных пластин (камертон)
   - Калибровка весовых площадок (эталонные грузы)
   - Синхронизация барометров (давление)

2. Настройка чувствительности:
   - Регулировка усиления рычажных систем
   - Установка демпфирования для устойчивости
   - Настройка порогов срабатывания
   - Балансировка всех подвижных частей

3. Испытания системы:
   - Тест с контролируемыми воздействиями
   - Проверка линейности всех каналов
   - Измерение скорости отклика
   - Долговременные испытания стабильности
```

### Периодическое обслуживание:

```bash
Ежедневно:
- Проверка натяжения тросов и струн
- Смазка критических подшипников
- Очистка от пыли резонансных элементов
- Проверка показаний контрольных приборов

Еженедельно:
- Калибровка по эталонным сигналам
- Регулировка демпфирования
- Проверка механических соединений
- Настройка резонансных частот

Ежемесячно:
- Капитальная смазка всех механизмов
- Замена изношенных струн и пружин
- Полная очистка и полировка
- Документирование изменений настроек
```
