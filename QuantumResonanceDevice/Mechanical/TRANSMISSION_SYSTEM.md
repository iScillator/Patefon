# ⚙️ Mechanical Transmission System - Система Механической Передачи

## Принципы механической обработки сигналов

Система полностью механической передачи и обработки информации основана на принципах **аналоговых вычислений**:

### 🔧 Основные элементы передачи:
1. **Дифференциальные механизмы** — сложение и вычитание сигналов
2. **Планетарные передачи** — масштабирование и усиление
3. **Кулачковые механизмы** — нелинейные преобразования
4. **Интеграторы** — накопление и усреднение
5. **Гармонические анализаторы** — частотное разложение

---

## 🎯 Центральный вычислительный механизм

### Механический аналоговый компьютер:

```
АРХИТЕКТУРА ВЫЧИСЛИТЕЛЬНОГО БЛОКА:

Основа - дифференциальный механизм Бэббиджа:
├── Входные валы: 32 канала от датчиков
├── Промежуточные валы: 64 канала обработки
├── Выходные валы: 16 каналов управления
├── Шестеренчатые передачи: >500 зубчатых колес
├── Материал шестерен: закаленная сталь + латунь
├── Точность изготовления: ±0.01 мм
└── Смазка: спермацетовое масло (как в хронометрах)

Схема обработки сигналов:

    ДАТЧИКИ (32 входа)
         │
    ┌────▼────┐
    │ ВХОДНОЙ │ ← Нормализация и фильтрация
    │ КАСКАД  │
    └────┬────┘
         │
    ┌────▼────┐
    │ОСНОВНОЙ │ ← Сложение, вычитание, умножение
    │ПРОЦЕССОР│   интегрирование, дифференцирование
    └────┬────┘
         │
    ┌────▼────┐
    │ВЫХОДНОЙ │ ← Формирование управляющих сигналов
    │ КАСКАД  │
    └────┬────┘
         │
    ИСПОЛНИТЕЛЬНЫЕ МЕХАНИЗМЫ (16 выходов)
```

### Дифференциальные механизмы:

```
ДИФФЕРЕНЦИАЛ ДЛЯ СЛОЖЕНИЯ СИГНАЛОВ:

Конструкция:
├── Входные валы: 2 (сигналы A и B)
├── Выходной вал: 1 (сигнал A+B или A-B)
├── Передаточное отношение: настраиваемое 1:0.1 до 1:10
├── Материал: закаленная сталь, шлифовка 6 класса
├── Люфт: <0.01° (практически отсутствует)
└── Точность: ±0.1% от входного сигнала

Схема дифференциала:
         Вход A
           │
           ▼
    ┌──────●──────┐ ← Планетарная шестерня
    │      │      │
    │   ┌──┴──┐   │ ← Сателлиты (4 шт)
    │   │     │   │
    │   └──┬──┘   │
    │      │      │
    └──────●──────┘ ← Выходное кольцо
           ▲
         Вход B

Результат: Выход = (A + B) / 2 или (A - B) / 2
```

### Интеграторы и дифференциаторы:

```
МЕХАНИЧЕСКИЙ ИНТЕГРАТОР (тип планиметра):

Принцип работы:
├── Диск интегратора: диаметр 200 мм, сталь
├── Колесо переменного радиуса: 0-100 мм от центра
├── Точность интегрирования: ±0.5%
├── Диапазон интегрирования: 0-360° (полный оборот)
├── Время интегрирования: настраиваемое 1-60 минут
└── Сброс: механический, ручной или автоматический

Схема интегратора:
    Входной сигнал (угол поворота диска)
              │
              ▼
    ┌─────────●─────────┐ ← Диск интегратора
    │         │         │
    │    ●────┼────●    │ ← Колесо переменного радиуса
    │    │    │    │    │   (позиция = входной сигнал)
    │    │    ●    │    │ ← Фрикционное колесо
    │    │   /│\   │    │   (обороты = интеграл)
    │    │  / │ \  │    │
    └────┴─┴──┴──┴─┴────┘
              │
              ▼
         Выходной вал (интеграл)
```

## 🎼 Спектральный анализатор

### Механический гармонический анализатор:

```
АНАЛИЗАТОР ФУРЬЕ (по принципу Михельсона):

Конструкция:
├── Количество гармоник: 88 (как у рояля)
├── Диапазон частот: 27.5 Гц - 4186 Гц
├── Принцип: механическое разложение сложного сигнала
├── Время анализа: 30 секунд на полный спектр
├── Точность: ±2% амплитуды каждой гармоники
└── Выход: 88 независимых сигналов амплитуды

Принцип работы:
1. Сложный входной сигнал подается на главный вал
2. 88 кулачковых механизмов настроены на разные частоты
3. Каждый кулачок "фильтрует" свою частоту
4. Планиметры интегрируют амплитуду каждой частоты
5. Результат - механический спектр сигнала

Схема одного канала анализатора:
    Входной сигнал
         │
         ▼
    ┌────●────┐ ← Кулачок-фильтр (настроен на частоту f)
    │    │    │
    │ ┌──┴──┐ │ ← Рычаг-детектор
    │ │     │ │
    │ └──┬──┘ │
    │    │    │
    └────●────┘ ← Интегратор амплитуды
         │
         ▼
    Амплитуда частоты f
```

### Частотные фильтры:

```
МЕХАНИЧЕСКИЕ ПОЛОСОВЫЕ ФИЛЬТРЫ:

Резонансные фильтры (88 каналов):
├── Каждый настроен на музыкальную ноту
├── Добротность: Q = 50-100
├── Материал резонаторов: закаленная сталь
├── Демпфирование: регулируемое масляными амортизаторами
├── Селективность: ±2% от центральной частоты
└── Динамический диапазон: 40 дБ

Принцип фильтрации:
- Входной сигнал возбуждает все резонаторы
- Каждый резонатор отвечает только на "свою" частоту
- Амплитуда отклика пропорциональна силе частотной компоненты
- Механические детекторы измеряют амплитуду каждого резонатора
```

## 🔗 Система передачи данных

```

Архитектура:

├── Третичные валы: 64 штуки, диаметр 12 мм

├── Подшипники: шариковые, класс точности 5
├── Муфты: кулачковые, быстроразъемные
└── Общая длина валов: >200 метров
```

## 🎚️ Управляющие механизмы

### Сервоприводы механические:

```
МЕХАНИЧЕСКИЙ СЕРВОПРИВОД (16 штук):

Конструкция:
├── Силовой элемент: пружинный аккумулятор энергии
├── Управляющий элемент: малое усилие от вычислителя
├── Усилитель мощности: рычажная система 1:50
├── Обратная связь: механический энкодер
├── Точность позиционирования: ±0.1°
├── Скорость отработки: 1 оборот/секунда
└── Сила на выходе: до 500 Н

Принцип работы:
1. Вычислитель выдает слабый управляющий сигнал
2. Управляющий механизм открывает пружинный аккумулятор
3. Энергия пружины через рычаги приводит в движение нагрузку
4. Обратная связь контролирует точность позиционирования
5. При достижении цели управляющий сигнал закрывает аккумулятор

Схема сервопривода:
    Управляющий сигнал (слабый)
              │
              ▼
    ┌─────────●─────────┐ ← Управляющий клапан
    │         │         │
    │    ┌────┴────┐    │ ← Пружинный аккумулятор
    │    │▓▓▓▓▓▓▓▓▓│    │   (накопленная энергия)
    │    └────┬────┘    │
    │         │         │
    │    ┌────┴────┐    │ ← Рычажный усилитель
    │    │   1:50  │    │   (усиление мощности)
    │    └────┬────┘    │
    │         │         │
    └─────────●─────────┘
              │
              ▼
         Выходной вал (мощный)
              │
              ▼ ← Обратная связь
    ┌─────────●─────────┐
    │   ЭНКОДЕР ПОЗИЦИИ │
    └───────────────────┘
```

## 🎵 Воздействие на звуковой тракт

### Управление тонармом:

```
СИСТЕМА УПРАВЛЕНИЯ ТОНАРМОМ:

Параметры управления:
├── Вертикальная сила прижима: 1-5 грамм
├── Антискейтинг: компенсация тангенциальной силы
├── Высота тонарма: ±5 мм
├── Азимут: ±2°
├── Скорость воспроизведения: 33⅓ ±5%
└── Демпфирование: регулируемое

Механизмы управления:
1. Регулировка прижимной силы - противовес на микрометрической резьбе
2. Антискейтинг - пружинная система с переменным плечом
3. Высота - эксцентриковый механизм с фиксацией
4. Азимут - микрометрический поворотный механизм
5. Скорость - центробежный регулятор с переменными грузиками

Схема управления тонармом:
    Команды от вычислителя
              │
    ┌─────────▼─────────┐
    │ БЛОК УПРАВЛЕНИЯ  │ ← Обработка команд
    │     ТОНАРМОМ     │
    └─────────┬─────────┘
              │
    ┌─────────▼─────────┐
    │     5 ПРИВОДОВ   │ ← Исполнительные механизмы:
    │ ├─ Прижим        │   - регулировка всех параметров
    │ ├─ Антискейтинг  │   - точная настройка звучания
    │ ├─ Высота        │   - компенсация износа
    │ ├─ Азимут        │   - оптимизация трекинга
    │ └─ Демпфирование │
    └─────────┬─────────┘
              │
              ▼
         ТОНАРМ С ИГЛОЙ ← Воспроизведение с пластинки
```

```
СИСТЕМА УПРАВЛЕНИЯ ЗАПИСЬЮ:

Параметры записи:
├── Глубина записи: 0.01-0.1 мм
├── Скорость движения иглы: 0.1 мм/оборот
├── Давление записи: 10-50 грамм
├── Частотная характеристика: 20 Гц - 15 кГц
├── Динамический диапазон: 40 дБ

Механизм записи:

4. Скорость записи синхронизирована с воспроизведением
5. Автоматическая смена восковых цилиндров

Схема записывающего механизма:
    Сигнал от резонаторов
              │
    ┌─────────▼─────────┐

              │
    ┌─────────▼─────────┐
    │ ЗАПИСЫВАЮЩАЯ     │ ← Вырезание канавки в воске
```

## 🔄 Система автоматического управления

### Алгоритмы обратной связи:

```
МЕХАНИЧЕСКИЙ РЕГУЛЯТОР (ПИД-контроллер):

Компоненты:
├── Пропорциональная составляющая (П): мгновенная ошибка
├── Интегральная составляющая (И): накопленная ошибка  
├── Дифференциальная составляющая (Д): скорость изменения ошибки
├── Коэффициенты: механически настраиваемые
├── Время переходного процесса: 1-5 секунд
└── Точность регулирования: ±1% от уставки

Реализация ПИД-регулятора:
1. П-составляющая: прямое рычажное соединение (ошибка → выход)
2. И-составляющая: механический интегратор (накопление ошибки)
3. Д-составляющая: дифференциатор (производная ошибки)
4. Сумматор: дифференциальный механизм объединяет все составляющие
5. Выходной усилитель: приводит в действие исполнительный механизм

Схема ПИД-регулятора:
    Уставка ────────┐
                    │
                ┌───▼───┐
    Обратная    │ СУММА-│  П-составляющая
    связь ──────┤ ТОР   ├─────────────┐
                │ ОШИБКИ│             │
                └───┬───┘             │
                    │                 │
        ┌───────────┼───────────┐     │
        │           │           │     │
        ▼           ▼           ▼     ▼
    ┌───────┐   ┌───────┐   ┌───────┐ │
    │   П   │   │   И   │   │   Д   │ │
    │состав-│   │состав-│   │состав-│ │ ← Три канала обработки
    │ляющая │   │ляющая │   │ляющая │ │
    └───┬───┘   └───┬───┘   └───┬───┘ │
        │           │           │     │
        └───────────┼───────────┘     │
                    │                 │
                ┌───▼───┐             │
                │ВЫХОДНОЙ            │ ← Окончательный
                │СУММАТОР├─────────────┘   управляющий сигнал
                └───┬───┘
                    │
                    ▼
            ИСПОЛНИТЕЛЬНЫЙ МЕХАНИЗМ
```

### Адаптивные алгоритмы:

```
САМОНАСТРАИВАЮЩАЯСЯ СИСТЕМА:

Принципы адаптации:
├── Обучение на статистике прошлых событий
├── Автоматическая подстройка параметров
├── Компенсация старения и износа механизмов
├── Адаптация к акустике конкретного помещения  
├── Учет сезонных изменений (температура, влажность)
└── Персонализация под предпочтения аудитории

Механизм обучения:
1. Система записывает результаты своих действий
```
